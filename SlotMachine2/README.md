<B>パラメータ</B></br>

| 項目 | 内容 |
| :--- | :--- |
|コインID|所持コインを保持する変数のIDを設定します|
|判定タイプ|スロットの判定タイプを設定します</br>左から判定：左のリールが揃っているかを判定</br>左右から判定：左右のどちらかのリールが揃っているかを判定</br></br>※5スロット用の設定です|
|7の獲得倍率|7のアイコンが揃った場合の獲得倍率を設定します|
|3スロットアイコン情報|リールに使用する3スロット用のアイコン情報を設定します|
|5スロットアイコン情報|リールに使用する5スロット用のアイコン情報を設定します|
|リールの描画位置(3スロット)|3スロット用のリールの位置を設定します|
|リールの描画位置(5スロット)|5スロット用のリールの位置を設定します|
|リールの回転速度(3スロット)|3スロット用のリールの回転速度を設定します</br>設定値を小さくすると遅くなります</br>設定値を大きくすると速くなります|
|リールの回転速度(5スロット)|5スロット用のリールの回転速度を設定します</br>設定値を小さくすると遅くなります</br>設定値を大きくすると速くなります|
|3スロットベットボタンの描画位置|3スロット用のベットボタンの位置を設定します|
|5スロットベットボタンの描画位置|5スロット用のベットボタンの位置を設定します|
|3スロットスタート/ストップボタンの描画位置|3スロット用のスタート/ストップボタンの位置を設定します|
|5スロットスタート/ストップボタンの描画位置|5スロット用のスタート/ストップボタンの位置を設定します|
|3スロット退出ボタンの描画位置|3スロット用の退出ボタンの位置を設定します|
|5スロット退出ボタンの描画位置|5スロット用の退出ボタンの位置を設定します|
|3スロットコイン情報描画タイプ|3スロット用のコイン情報の表示方法を設定します</br>横並び：左にラベル右にコインの枚数</br>縦並び：上にラベル下にコインの枚数|
|5スロットコイン情報描画タイプ|5スロット用のコイン情報の表示方法を設定します</br>横並び：左にラベル右にコインの枚数</br>縦並び：上にラベル下にコインの枚数|
|3スロットコイン情報長さ|3スロット用のコイン情報の長さを設定します|
|5スロットコイン情報長さ|5スロット用のコイン情報の長さを設定します|
|3スロットコイン情報描画位置|3スロット用のコイン情報の位置を設定します|
|5スロットコイン情報描画位置|5スロット用のコイン情報の位置を設定します|
|3スロットコイン情報倍率|3スロット用のコイン情報の拡大倍率を設定します</br>倍率は１が基準となります|
|5スロットコイン情報倍率|5スロット用のコイン情報の拡大倍率を設定します</br>倍率は１が基準となります|
|勝利メッセージ|揃った場合に表示されるメッセージを設定します</br>"%1"は獲得したコイン"%2"はコインの単位に置換されます。|
|敗北メッセージ|揃わなかった場合に表示されるメッセージを設定します|
|コイン不足メッセージ|コインが不足している場合に表示されるメッセージを設定します|
|コインの単位|コインの単位を設定します|
|勝利ME|揃った際に流すME情報を設定します|

<B>プラグインコマンド（引数）</B></br>

| 項目 | 内容 |
| :--- | :--- |
|スロットタイプ|開始するスロットのタイプを設定します</br>3スロット(3ライン)：3ライン用の3スロットを開始します</br>3スロット(5ライン)：5ライン用の3スロットを開始します</br>5スロット：5スロットを開始します|
|倍率|スロットの倍率(レート)を設定します|
|背景タイプ|背景のタイプを設定します</br>カラーコード：背景をカラーコードで設定</br>RGB：背景をRGBで設定</br>画像：背景画像から選択して設定|
|背景色(カラーコード)|カラーコード用の背景色を設定|
|背景色(RGB)|RGB用の背景色を設定|
|背景ファイル|背景に使用する画像を設定|
|レートオブジェクト|レートオブジェクトの表示方法を設定</br>全表示：オブジェクトを全て表示</br>レートのみ表示：倍率(レート)の部分だけ表示</br>非表示：オブジェクトを表示ません|
|リールの順番(3スロット)|3スロット用のリールの順番を設定します</br>※リールの数は統一しなくても大丈夫ですが、最低5個以上は設定してください|
|リールの順番(5スロット)|5スロット用のリールの順番を設定します</br>※リールの数は統一しなくても大丈夫ですが、最低5個以上は設定してください|
